Security Layer — Sensor Data Validation

Attack buttons send readings straight into the live intersection. With Security ON, every reading is checked (rule-based + Isolation Forest) before it's allowed to change the traffic light's view of the road — rejected readings never reach the controller. Toggle Security OFF to see what an attacker could actually do to the light with no validation layer in front of it.
